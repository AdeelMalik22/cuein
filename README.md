# Cuein

Cuein is a lightweight lead follow-up tool for small service businesses. Its purpose is simple: help a team remember every customer and every next follow-up.

This project will be built from scratch, step by step, with an emphasis on understanding how a multi-tenant Django application works.

## Product direction

- Each business has its own users and data.
- Salespeople track leads, activities, and follow-up tasks.
- Owners and managers can see their whole business.
- A business must never be able to see another business's data.

The detailed product scope is in `PRD-FollowUpCRM.md`, and the current delivery plan is in `plan.md`.
